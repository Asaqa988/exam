Manual Testing – MCQs
1. What is the primary objective of manual testing?

A. To eliminate the need for automation
B. To verify software meets specified requirements
C. To write test scripts
D. To reduce development cost

✅ Correct Answer: B

2. Which testing type is usually performed first?

A. Regression Testing
B. System Testing
C. Unit Testing
D. Acceptance Testing

✅ Correct Answer: C

3. Which document describes what needs to be tested?

A. Test Case
B. Test Plan
C. Requirement Specification
D. Defect Report

✅ Correct Answer: C

4. What is Smoke Testing mainly used for?

A. Checking all edge cases
B. Verifying critical functionalities after a build
C. Performance testing
D. Validating UI design

✅ Correct Answer: B

5. When should Regression Testing be performed?

A. After requirement gathering
B. After every code change or bug fix
C. Only before release
D. Only during UAT

✅ Correct Answer: B

6. What is a test case?

A. A bug description
B. A set of steps with expected results to validate functionality
C. A project requirement
D. A test environment

✅ Correct Answer: B

7. Which testing is done without executing the code?

A. Functional Testing
B. Dynamic Testing
C. Static Testing
D. Regression Testing

✅ Correct Answer: C

8. Which scenario is best suited for Exploratory Testing?

A. When requirements are very detailed
B. When time is limited and documentation is minimal
C. When automation is mandatory
D. When performance metrics are required

✅ Correct Answer: B

9. Which testing ensures that new changes have not broken existing functionality?

A. Smoke Testing
B. Sanity Testing
C. Regression Testing
D. Integration Testing

✅ Correct Answer: C

10. Which defect priority indicates that the issue must be fixed immediately?

A. Low
B. Medium
C. High
D. Critical

✅ Correct Answer: D

11. Which testing type is performed by end users or clients?

A. System Testing
B. Integration Testing
C. User Acceptance Testing (UAT)
D. Regression Testing

✅ Correct Answer: C
